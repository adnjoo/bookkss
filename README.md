Front-end to Bookkss app that provides a platform for users to write and share book reviews.

The app is [deployed](https://bookkss.com) with Vercel

---

Technologies used:

Front-end

- Next 13.x
- React 18.x
- TypeScript 5.x

Middleware

- Next-Auth 4.x

---

Quick start (client):

```
npm run dev
```
