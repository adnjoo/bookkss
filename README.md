Bookkss provides a platform for users to write and share book reviews.

[@FE](https://bookkss.com) Vercel

[@DB](https://supabase.com/dashboard) Supabase x Postgres

[@BE](https://bookkss.fly.dev) Fly

---

Technologies used:

Front-end

- Next 13.x
- React 18.x
- TypeScript 5.x

Middleware

- Next-Auth 4.x

Back-end

- Express 4.x

---

Quick start (client):

```
cd client
npm run dev
```

Quick start (server):

```
cd server
npm install
npm run dev
```
